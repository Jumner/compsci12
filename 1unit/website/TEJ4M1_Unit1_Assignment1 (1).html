The aim of this assignment is to learn how structure and presentation of web content can be separated using HTML5 and CSS.

Task 1: Learn HTML5 and CSS

Learn to write HTML5 and Cascading Style Sheets (CSS) by following the tutorials at http://https://www.w3schools.com/html/default.asp/ and http://www.w3schools.com/css/.

Task 2: Create Web Pages [ /18]

Create a web page on some topic that you know something about.
You may use embedded or external style sheets to format the appearance of your web page.
If you wish, in the style of Zen Garden, you may use Javascript to allow the user to determine which style your web site will be rendered in.
Your program should should fulfill the following criteria:

  • Comment your code appropriately
  • Use at least one image and one table.
  • Define at least three different CSS classes in your style sheet
  • Use generic style classes, multiple selectors and pseudo-elements
  • Read up on CSS pseudo-classes and define two classes of hyperlinks: one where the links turn green when the mouse hovers over them and one where the links turn red.
  • Make sure you separate structure and presentation as much as possible.
  • Use indentation to make sure your code is readable and make sure you have all the closing tags in place.

Task 3: Validate Your Pages [ /12]

Use the World Wide Web Consortium's (W3C) HTML validation service at http://validator.w3.org/ to check that your page conforms to the HTML and CSS standards. You can either upload the page to the W3C or put it online and type the URL into the W3C's validator form.
